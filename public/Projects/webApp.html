Web Application For Blood Bank System Completion Year : June 2018 - April 2019
Project Description

The Main Aim of Developing this system is to provide blood to the people who are in need of blood. Using this system user can easily search the donor and their blood group Available in the city and he can also get contact number of donor who has the same blood group.

Technology Stack
  • Front End

    Html5 , CSS3 , Bootstrap-4 , Java Script

  • Back End

    Php

  • Database

    Mysql
